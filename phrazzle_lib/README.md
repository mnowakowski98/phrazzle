
# Phrazzle

A game about finding subsets of words and phrases from words and phrases.

## Game Rules

- Number of players: 1 - ∞ (but you probably want at least 2).
- Number of rounds: Arbitrary af.

### Round flow

- Choose a starting phrase.
> Example: the quick brown fox jumps over the lazy dog
- From the starting phrase create smaller phrases using the same letters.
> Example: heck jog

In derived phrases:  

- Phrase doesn't necessarily have to make sense.
- Don't just steal whole ass word(s) from the starting phrase, it do be boring.
- Order of letter occurance must be preserved.
- Capitalization and spaces do not have to be preserved.

> Invalid derived phrase examples:  
> - fox jumps  
> - heart


- Think of more phrases from the previous phrase.
> Example: hog
- The number of derived phrases is your score for the round.
- Next player uses the same starting phrase and trys to think of unique derivations.  
> \* Maybe do this simultaneously.
